# 5 如何进化

构建先进、自主且日益通用的人工智能，其核心追求在于自进化。对于大型语言模型（Large Language Models，LLMs）及其智能体扩展而言，如何持续、自主且高效地进化其能力，已成为一个核心挑战。因此，自进化智能体的第三个关键方面，便是实例化一个有效的进化策略 $f$，即如何将一个智能体系统 $\Pi = (\Gamma, \{\psi_i\}, \{C_i\}, \{W_i\})$ 转化为其新状态 $\Pi' = (\Gamma', \{\psi_i'\}, \{C_i'\}, \{W_i'\})$。与传统方法（依赖于静态数据集或一次性监督微调）不同，自进化强调的是一个持续进行的过程：模型从真实世界交互中学习，主动寻求反馈，进行自我反思，生成或筛选新数据，并根据动态环境调整策略。这种持续进化不仅仅是扩大数据量或计算量的问题；它要求智能体获得一系列元能力（meta-capabilities），包括自我修正、自主数据生成、知识迁移和多智能体协作。因此，自进化的图景变得日益丰富和多维度，每个方法论分支都在探索反馈、学习范式、数据源和进化规模等不同维度上的可能性。

随着时间推移，自进化智能体研究经历了三大范式——基于奖励的进化、基于模仿的进化和基于种群的进化——每一范式都是为了解决前一范式的局限性而兴起的。基于奖励的方法首先通过显式信号闭合了反馈回路，但存在脆弱性高和成本昂贵的问题。基于模仿的学习通过利用高质量的示范数据稳定了进化过程，尽管有时以牺牲探索性为代价。基于种群的进化则将适应扩展到群体层面，强调多样性和涌现式协作。这些范式共同描绘了一条从个体自我提升到集体智能的连贯轨迹。

本章旨在系统地梳理和分析自进化方法的主要类别，提供一个统一框架以理解其原理、机制和交互关系。我们首先介绍基于奖励的进化，其核心在于奖励信号的设计——从自然语言反馈、内部置信度指标到外部或隐式信号——以引导迭代式自我改进。

## 5.1 基于奖励的自进化

自我提升能力是高级智能的基石。在大型语言模型（LLMs）的语境下，这表现为一种动态的奖励驱动进化过程，模型在此过程中迭代地从自身的输出和交互中学习，以精炼其能力。奖励信号作为引导性反馈，其设计至关重要；它决定了学习过程的性质、效率和效果。在本节中，我们系统地回顾奖励设计的主要方法，按反馈的性质分为：文本反馈、内部置信度、外部奖励和隐式奖励。

### 文本反馈

文本反馈利用大型语言模型（LLMs）的原生模态——自然语言——来提供详细且可解释的改进指令。与标量奖励不同，文本反馈蕴含着细腻的批评和可操作的建议。近期的框架，如 Reflexion（Shinn 等，2023）、AdaPlanner（Sun 等，2023）、AgentS2（Agashe 等，2025）、SELF（Lu 等，2023）、Self-Refine（Madaan 等，2023a）、SCoRe（Kumar 等，2024）、PAG（Jiang 等，2025b）和 TextGrad（Yellamraju 等，2024），都是这一方向的典型代表。例如，Reflexion 提出了"言语强化学习"，其中智能体以自然语言对过往尝试进行反思，并将这些反思存储为情景记忆，以指导未来的决策。AdaPlanner 通过允许 LLM 智能体根据计划内和计划外反馈来修订计划，实现闭环自适应规划，同时通过代码风格的提示词减轻幻觉，并利用技能发现。Self-Refine 和 SELF 进一步探索了迭代式自我反馈和自我修正，表明即使是最先进的模型，也可以通过多轮基于语言的自批评来改进，无需额外的监督数据或外部强化。这些框架彰显了语言作为奖励通道的力量，使细致、灵活且样本高效的自我提升成为可能。

### 内部奖励

基于内部置信度的奖励方法不依赖外部信号，转而利用模型自身的概率估计或确定性等内部指标。这一范式利用模型的内生理解来引导改进，而不依赖于外部监督。包括置信度知情自洽性（Confidence-Informed Self-Consistency，CISC）（Taubenfeld 等，2025）、自集成（Self-Ensemble）（Xu 等，2025b）、自我奖励自我改进（Self-Rewarding Self-Improving）（Simonds 等，2025）、通过自确定性实现可扩展的最佳选择（Kang 等，2025）以及自我奖励语言模型（Self-Rewarding Language Models）（Yuan 等，2025b）等方法，使模型能够基于内部置信度指标进行自我评估和响应校准。例如，CISC 通过置信度分数对推理路径进行加权，同时提高准确率和计算效率，有效地从一个候选解集中筛选出高质量解。自集成通过将选项划分为更小、更易管理的组，并聚合预测结果来减轻置信度失真，从而减少过度自信的偏差。自我奖励语言模型证明了模型可以作为自身的奖励函数，通过自我指令和自我评估循环生成训练数据。这些方法可以减少对人类标签和外部评估者的依赖，使可扩展且自主的自我提升循环成为可能，无需人类干预即可持续运行。AgentEvolver（Zhai 等，2025）提出了一个综合框架，通过三种协同机制提高智能体训练效率：用于自主任务生成的自我提问、用于经验引导探索的自我导航，以及用于细粒度信用分配的自我归因。其自我归因机制利用 LLM 的推理来回顾性地分配逐步奖励，这些奖励对于策略优化而言是密集且语义上有依据的。

### 外部奖励

外部奖励来源于模型外部的因素，如环境、多数投票或显式规则。多数投票（Shafayat 等，2025；Wei 等，2025c；Zhang 等，2025j）利用多个模型输出之间的共识作为正确性的代理，提供一种自生成但有依据的奖励信号。环境反馈（包括基于工具的信号）在智能体 LLM 研究中处于核心地位（如 SWE-Dev（Du 等，2025）、SICA（Robeyns 等，2025a）、Feedback Friction（Jiang 等，2025a）、USEagent（Applis 等，2025）、DYSTIL（Wang 等，2025c）），智能体通过与真实世界环境和工具的直接交互来学习。基于规则的奖励（Wang 等，2025h；Wang & Xiong，2025；Zhang 等，2025a；Simonds & Yoshiyama，2025；Wang 等，2025q；Liu 等，2025d）使用显式约束或逻辑规则作为可验证的信号，在数学推理、游戏博弈和结构化问题解决等领域尤为有效。这些方法提供了客观、可靠的监督，但可能需要大量工程工作或在表达能力上存在局限。

### 隐式奖励

隐式奖励框架假设大型语言模型（LLMs）可以从并非显式标注为奖励的反馈信号中学习。例如，"奖励已经足够"（Reward Is Enough）（Song 等，2025）表明 LLM 可以使用嵌入在上下文窗口中的简单标量信号进行上下文强化学习，在多轮对话中改进其响应，无需显式的强化学习微调或监督。这揭示了模型在输入上下文中解释和学习隐式反馈线索的内在能力。近期的工作通过表明 LLM 通过其标准训练目标内在地编码了类似奖励的信号，将这一概念进一步扩展。内生奖励（Endogenous Reward）（Li 等，2025e）揭示了标准的下一个-token 预测隐式地学习了一个通用奖励函数，可以无需额外训练直接从模型 logit 中提取。此外，隐式自我改进框架（ImPlicit Self-ImprovemenT，PIT）（Wang 等，2024i）通过最大化在参考响应条件下响应质量差距来隐式地从人类偏好数据中学习改进目标，无需额外人力。与基于规则的或源自环境的外部奖励不同，隐式奖励方法的独特优势在于发现并利用语言建模中内在存在的奖励信号。

综上所述，基于奖励的进化提供了显式优化和强大的自主性，但对奖励设计较为敏感，往往以稳定性和安全性换取适应性和开放性。
