# 6 何处进化？

自进化智能体推动了众多领域和应用的进展。广义上，这些应用可以系统性地分为两大类：(1) 通用领域进化（General Domain Evolution），即智能体系统通过扩展其在多种任务中的能力来实现进化，主要发生在数字领域；(2) 专业领域进化（Specialized Domain Evolution），即智能体在特定任务领域内深化其能力。从本质上讲，通用助手的进化侧重于将习得的经验迁移到更广泛的任务集合中，而专业智能体的进化则强调在特定领域内深耕专业知识。

## 6.1 通用领域进化

第一类为通用领域进化，指的是为通用应用场景（尤其是作为多功能数字助手）而设计的自进化智能体。这类智能体逐步提升其能力，以应对广泛的用户查询，特别是在动态多样的数字环境中。从技术层面看，通用助手智能体主要通过三种机制来增强自身能力：记忆优化（Memory Mechanism）、课程驱动训练（Curriculum-Driven Training）和模型-智能体协同进化（Model-Agent Co-Evolution）。这些机制共同使智能体能够持续适应并有效回应日益复杂的用户需求。

**记忆机制（Memory Mechanism）**。促进智能体进化的最常见机制是记忆机制。智能体将历史成功/失败经验（Wang et al., 2023a; Zhang et al., 2024f）总结为记忆表征（Zhang et al., 2024g），期望这些提炼的经验在处理未曾见过的任务时能发挥作用。例如，Mobile-Agent-E（Wang et al., 2025o）采用了一种长期记忆结构，由"提示（Tips）"——提供一般性指导原则——和"快捷方式（Shortcuts）"——从过往经验中提取的可复用动作序列——组成。这一自进化模块支持在复杂智能手机任务上实现性能的持续提升。另一个典型例子是 MobileSteward（Liu et al., 2025f），它在中心智能体的协调下管理多个针对特定应用的智能体，并配有专门的任务调度、执行和评估模块。该系统还引入了一种基于记忆的自进化机制，通过总结成功执行案例来改进未来跨应用指令的处理。与此同时，生成式智能体（Generative Agents）（Park et al., 2023）存储其经历的情景记忆，综合出更高层次的反思，并在此自我反思的基础上条件化未来的规划。在这些案例中，记忆作为基础使智能体能够将过往经验内化，抽象出高层模式，并优化其未来行为。

**模型-智能体协同进化（Model-Agent Co-Evolution）**。另一条研究路线是针对大语言模型智能体执行模型-智能体协同进化。UI-Genie（Xiao et al., 2025a）构建了一个专门的图文奖励模型，在步骤级和任务级两个层面对轨迹进行评分。它使用由受控数据腐蚀（controlled corruption）和困难负样本挖掘（hard-negative mining）生成的合成轨迹，联合微调智能体和奖励模型。WebEvolver（Fang et al., 2025b）引入了一个协同进化的世界模型大语言模型（world model LLM），用于模拟网络环境。它通过预测下一观测结果来生成合成训练数据，并在推理过程中实现前瞻推理（look-ahead reasoning），显著提高了真实网络任务的成功率。Absolute Zero（Zhao et al., 2025a）通过强化自我对弈（reinforced self-play）使推理智能体及其内部自奖励模型协同进化。该框架通过对抗性地生成越来越具有挑战性的推理问题，并使用内部自我确定性（internal self-certainty）作为奖励信号来优化智能体，从而同时更新智能体的策略和自奖励机制。综上所述，这些方法证明了协同进化智能体与辅助模型（如奖励模型或世界模型）对于在语言模型智能体系统中实现更稳健、更具泛化能力和可扩展性的学习的有效性。

**课程驱动训练（Curriculum-Driven Training）**。课程驱动训练也是构建自进化通用助手的关键机制。例如，WebRL（Qi et al., 2024）采用自进化课程机制：当智能体失败时，系统自动生成相似但可管理的任务。结合习得的奖励模型和自适应策略更新，该方法在 WebArena 基准上实现了成功率提升。Voyager（Wang et al., 2023a）同样在 Minecraft 中利用自动化的自底向上课程，其中 GPT-4 根据智能体进度提出适当的下一个任务，通过迭代提示和环境反馈构建不断扩充的基于代码的技能库。这些方法表明，课程学习如何使智能体能够通过迭代任务适应自主扩展其能力。

## 6.2 专业领域进化

除了通用数字智能体外，自进化智能体也有效应用于专业领域，在较窄的任务集合中显著提升其表现。

**编程（Coding）**。自进化智能体的能力直接延伸到编程等实际应用中，其自主适应与改进的能力为软件开发提供了一种变革性的方法。SICA（Robeyns et al., 2025a）证明了一个自改进编程智能体可以自主编辑自身代码库并在基准任务上提升表现。EvoMAC（Hu et al., 2024d）引入了一种多智能体协作网络上的自进化范式，自动优化单个智能体的提示词和多智能体工作流程，通过克服人工设计系统的局限性显著提升了代码生成性能。AgentCoder（Huang et al., 2024）同样聚焦于一个通过迭代改进实现自进化的多智能体代码生成框架。程序员智能体根据来自测试执行智能体的反馈持续改进代码，并基于测试设计智能体提供的独立测试用例进行验证，显著提高了有效性和效率。Zhang 等人（Zhang et al., 2025b）使大语言模型智能体能够通过筛选高质量答案、按难度分层积累经验，并从自生成数据中自适应选择演示来持续进化，从而实现了显著的性能提升并构建了机器学习库。虽然这些案例的具体机制各异——从单智能体自我编辑到复杂多智能体协作网络以及基于经验的学习——但它们共同遵循自主迭代改进和自适应这一核心原则，以提升编程能力。这些进展凸显了自进化智能体如何通过持续学习和优化显著提升编程效率和代码质量。

**图形用户界面（Graphical User Interfaces, GUI）**。自进化GUI智能体将大语言模型的能力从纯文本推理扩展到对桌面端、Web 端和移动端界面的直接操控，这些场景需要应对大尺度离散动作空间、异构布局和部分视觉可观测性等挑战。Yuan 等人将像素级视觉与自我强化相结合，使智能体能够迭代优化点击-输入的定位精度，无需额外人工标注（Yuan et al., 2025c）。在真实桌面软件上，来自 WindowsAgentArena 的 Navi 智能体重放并批判自身的失败轨迹，最终在 150 项 Windows 挑战中将任务完成率翻倍（Bonatti et al., 2024）。对于开放网络自动化，WebVoyager 将截图特征与思维链反思相融合；持续的自我微调使其在未见网站上的端到端成功率从 30% 提升至 59%（He et al., 2024），而 ReAP 添加了过去结果的情景记忆，在先前失败过的查询上进一步恢复了 29 个百分点的性能（Azam et al., 2025）。除强化学习和记忆机制外，AutoGUI 持续从实时界面中挖掘功能标注，以在每个训练周期扩充可复用的技能库（Li et al., 2025a），MobileUse 则部署了分层自我反思栈，在智能手机动作的执行现场进行监控、验证和修订（Li et al., 2025c）。总体而言，这些系统体现了自进化的完整三要素——进化的对象（定位模块、技能记忆）、进化的时机（离线整合 vs. 在线反思）、以及如何进化（强化学习、合成数据、分层监控）——并由此开辟了一条迈向通用能力界面智能体的道路。

**金融（Financial）**。为金融任务等专业领域定制智能体的主要瓶颈在于高效构建并将对领域特定知识库的整合融入智能体的学习过程——这一问题可以通过引入自进化机制得到有效缓解。QuantAgent（Wang et al., 2024d）提出了一个双层框架，通过模拟环境和真实环境的反馈迭代优化智能体的响应，并自动增强其领域特定知识库。这一迭代过程帮助智能体逐步逼近最优行为，降低对昂贵人工策划数据集的依赖，并在交易任务中明显提升了预测准确率和信号质量。TradingAgents（Xiao et al., 2024）整合了反思、强化学习和来自真实交易结果的反馈循环，以及协作辩论机制，持续优化策略并提升交易表现。这些进展表明，自进化智能体有潜力通过自主构建领域专业知识、适应动态市场条件以及持续改进决策和交易表现来革新金融领域。

**医疗（Medical）**。自进化智能体已成为医疗人工智能中的一个强大范式，其适应性和进化能力对于应对真实临床实践的复杂性和不断变化的特性至关重要。最突出的应用之一是医院级模拟。例如，Agent Hospital（Li et al., 2024a）创建了由大语言模型驱动的医生、患者和护士组成的封闭环境，使医生智能体能够治疗数千个虚拟病例。这一过程帮助这些智能体无需人工标注即可自主完善和进化诊断策略，最终在美国医学执照考试（USMLE）风格的考试中取得了优异成绩。类似地，MedAgentSim（Almansoori et al., 2025a）整合了大语言模型医生、患者和工具智能体。它将成功的会诊记录为可复用的轨迹，并利用思维链反思和共识机制来驱动自进化，在连续交互中提高成功率。另一个例子是 EvoPatient（Du et al., 2024），它将医生智能体和患者智能体置于持续对话中。每一代互动中，双方都会用高质量的对话内容更新记忆：患者生成更真实化的症状叙述，医生则学会提出更精准的问题。值得注意的是，这一过程无需显式的梯度更新或手工奖励。强化学习同样是构建自适应医疗智能体的核心。例如，DoctorAgent-RL（Feng et al., 2025c）将问诊建模为马尔可夫决策过程，使用对诊断准确率、覆盖率和效率评分的奖励函数。这引导了策略梯度更新，使智能体能够提出更相关的问题，并比基于模仿学习的方法更快地得出正确诊断，从而实现自我改进。此外，自动化架构搜索方法如"学会做医生"（Learning to Be a Doctor）将工作流本身视为一个可进化的对象，迭代地插入专科子智能体或新的推理步骤，以覆盖观察到的失败模式并提升多模态诊断准确率（Zhuang et al., 2025）。最后，除临床决策外，自进化智能体也已被拓展到生物医学发现领域。OriGene（Zhang et al., 2025t）作为一个虚拟疾病生物学家，通过迭代优化分析过程来实现进化。它利用人类反馈和实验反馈来更新核心推理模板、调整工具使用策略并完善分析流程。类似地，STELLA（Jin et al., 2025）是一个自进化的生物医学研究智能体，通过将其模板库（Template Library）中的成功推理工作流提炼为可复用模板，并扩展其工具海洋（Tool Ocean）以整合外部工具或新组装工具来满足新出现的分析需求，从而实现随时间的持续改进。

**教育（Education）**。自进化大语言模型智能体在教育领域也有着广泛的应用。在学习者层面，像个性化导师 PACE（Liu et al., 2025c）这样的自进化智能体会根据详细的学生画像调整提示词，并在对话过程中持续优化其提问方式。与此同时，一个大语言模型对大语言模型的自我对弈框架（LLM-to-LLM self-play framework）生成多样化的导师-学生对话，用于进一步微调智能体，使其教学策略在交互过程中和交互结束后都能持续进化。另一个例子是 MathVC（Yue et al., 2025），它使用符号化人格画像（symbolic persona profiles）为虚拟学生建模，并配备一个元规划器（meta-planner）来编排真实的解题阶段。这一设置使智能体的对话过程能够逐步向正确答案演进，紧密模拟协作学习的自然展开方式。在导师方面，像专业发展平台 i-vip（Yang et al., 2025b）这样的自进化智能体系统部署了一组协作的大语言模型智能体——教练、评估者和反馈生成器——它们实时地相互评判和改进输出。这些智能体根据教师-学习者的反馈调整其解释方式，并在部署后持续通过整合专家反馈进行进化，从而随时间优化其提示策略。类似地，EduPlanner（Zhang et al., 2025p）将教案创建框架设计为一个对抗性循环，其中规划者的草稿由评估智能体和优化智能体反复审查和精炼，直至满足多元教育目标。SEFL（Zhang et al., 2025k）同样使用师生自我对弈生成大量作业-反馈示例，然后用它们微调一个轻量级反馈模型。这一自进化过程显著提升了反馈的清晰度和有用性。总体而言，这些例子说明了自进化大语言模型智能体如何能动地适应学习者和教育者双方，推动更加个性化、有效和可扩展的教育体验。

**其他领域（Others）**。除上述四个主要垂直领域外，自进化智能体还展现出更广泛的应用潜力，在传统智能体往往表现不足的专业领域提供卓越的适应性和性能。例如，Arxiv Copilot（Lin et al., 2024）通过将历史用户交互——包括生成的回答、研究趋势和想法——纳入其思维数据库来进行学习和适应，从而增强其提供个性化和增强型学术辅助的能力。在另一个截然不同的场景中，Minecraft 游戏中的智能体 Voyager（Wang et al., 2023a）能够通过自进化过程在新世界中从零开始解决全新任务。它通过自动课程持续优化任务目标，扩充技能库，并利用迭代提示机制在无需人工干预的情况下增强其动作能力。在需要显式战略规划的其他领域，Agents-of-Change（Belle et al., 2025）基于迭代性能分析和战略研究自主优化提示词并重写代码，从而帮助智能体克服长期战略规划中的固有局限性，并在诸如《卡坦岛》（Settlers of Catan）等复杂环境中实现始终优异且更连贯的游戏表现。最后，在外交领域，Richelieu（Zhao et al., 2024d）引入了人工智能外交智能体，能够通过其自我对弈机制实现自进化，使智能体无需人工数据即可通过获取多样化经验来扩充记忆，从而增强其在外交活动中的战略规划、反思和整体表现。虽然这些多样化的例子运行在截然不同的环境中——从学术研究、虚拟游戏世界到战略棋盘游戏和复杂外交谈判——但它们都有一个共同的基本特征：利用持续学习、自我改进和自主适应，在各自领域中实现越来越复杂和有效的表现。这些多样化的例子进一步印证了自进化智能体的多功能性，展示了它们在超越传统领域的广泛复杂、动态和类人任务中日益增长的卓越潜力。
